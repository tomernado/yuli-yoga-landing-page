## Purpose

Provide security-focused code review and vulnerability analysis by leveraging an approved, well-maintained security resource instead of relying solely on ad-hoc manual review.

---

## Primary Resource

Name:

Claude Code Security Review

Repository:

https://github.com/anthropics/claude-code-security-review

---

## Why This Resource

This resource provides structured security review capabilities for software projects, including:

- Authentication and authorization review.
- Input validation.
- Secret detection.
- Common web security vulnerabilities.
- Security-focused code review.
- Secure engineering practices.
- Risk identification before deployment.

Prefer this resource over creating custom security review guidance from scratch.

---

## Use When

Use this resource for tasks involving:

- Security reviews.
- Authentication.
- Authorization.
- API security.
- Input validation.
- Production readiness.
- Sensitive functionality.
- Security validation before deployment.
- Reviewing pull requests for security risks.

---

## Do Not Use When

Do not use this resource for:

- UI design.
- Styling.
- Business logic implementation.
- Architecture documentation.
- General frontend development.
- Non-security tasks.
