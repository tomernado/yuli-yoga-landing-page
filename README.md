# Yuli Yoga — Landing Page

A single-page, RTL Hebrew marketing site for Yuli's yoga practice, built with React, Vite and TypeScript.

Project context and standards live in [`AI/`](AI/CLAUDE.md) — see [`AI/docs/PROJECT.md`](AI/docs/PROJECT.md) and [`AI/docs/ARCHITECTURE.md`](AI/docs/ARCHITECTURE.md) before making changes.

## Development

```bash
npm install
npm run dev      # start the dev server
npm run build    # type-check and build for production
npm run lint      # run oxlint
```
